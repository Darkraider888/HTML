**HTML** stands for **HyperText Markup Language**.  

- It is used to **create the structure of web pages**.  
- It is **not a programming language**.  
- HTML uses **tags** to describe content like text, images, links, and forms.

> By the end of this repository, you will be able to create **simple web pages** and understand the **basic building blocks** of HTML.

| Tag | Description |
|-----|-------------|
| `<!DOCTYPE html>` | Tells the browser that this is an HTML5 document |
| `<html>` | The root tag of the HTML page |
| `<head>` | Contains page information (not visible on the page) |
| `<title>` | Appears on the browser tab |
| `<body>` | Contains all visible content on the web page |
