### ðŸ”¹ What is a List?

A **list** is used to show multiple related items together.

HTML has **two main types of lists**:

1. **Unordered List** (bullets)
2. **Ordered List** (numbers)

---

### ðŸ”¹ Unordered List (`<ul>`)

Used when order does **not matter**.

Example:

```html
<ul>
    <li>Apple</li>
    <li>Banana</li>
    <li>Orange</li>
</ul>
```

---

### ðŸ”¹ Ordered List (`<ol>`)

Used when order **does matter**.

Example:

```html
<ol>
    <li>Wake up</li>
    <li>Brush teeth</li>
    <li>Go to class</li>
</ol>
```

---

### ðŸ”¹ List Items (`<li>`)

* `<li>` means **list item**
* Always used inside `<ul>` or `<ol>`

---

### ðŸ’¡ Beginner Tips:

* Use `<ul>` for normal lists
* Use `<ol>` for step-by-step lists
* Never use `<li>` outside a list
